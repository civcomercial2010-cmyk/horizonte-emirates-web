# Horizonte Emirates — Brand Assets

Carpeta de logotipos creada para uso local como activos de marca. Incluye versiones independientes con soporte fiel al mockup de barra elegante y construcción geométrica.

## Archivos incluidos

- `logo-horizontal.svg` — versión horizontal principal con símbolo dorado y texto navy.
- `logo-horizontal-dark.svg` — versión horizontal lista para fondo navy con texto blanco.
- `logo-symbol.svg` — símbolo independiente en dorado para aplicaciones compactas.
- `logo-symbol-dark.svg` — símbolo independiente sobre fondo navy.
- `logo-stack.svg` — versión apilada con símbolo encima del texto.
- `favicon.svg` — versión simplificada para favicon.

## Uso recomendado

- `logo-horizontal.svg` — header web claro, presentación de marca y materiales impresos sobre fondo crema.
- `logo-horizontal-dark.svg` — hero oscuro, anuncios display y piezas en fondo navy.
- `logo-symbol.svg` — favicon, avatar, aplicación icon y monograma independiente.
- `logo-stack.svg` — documentos de marca, tarjetas y piezas verticales.
- `favicon.svg` — favicon del sitio y icono digital.

## Colores

- Base:
  - `#303257` — navy profundo
  - `#E6E68C` — dorado suave
- Soporte:
  - `#F7F1E9` — crema arena
  - blanco para texto en fondos oscuros

## Observaciones

Estos activos están diseñados para reflejar fielmente el estilo del mockup: barras verticales geométricas, alineamiento técnico de texto y versión sobre fondo crema y navy. Los SVG son fáciles de ajustar en programa de vectores si se requiere mayor refinamiento tipográfico o offset de líneas.
